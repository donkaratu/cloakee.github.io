# README.md

## Progressive Web App Features
- Offline Support
- Responsive Design
- Add to Home Screen
- Push Notifications

## Installation Instructions

### Desktop
1. Open your browser and navigate to the website.
2. Click on the menu icon (usually at the top right).
3. Select 'Install' to add the application to your desktop.

### Mobile
1. Open your browser (Chrome/Safari) and navigate to the website.
2. Tap on the menu icon or the share button.
3. Select 'Add to Home Screen'.

## Keyboard Controls
- **Arrow keys**: Navigate through app components.
- **Enter**: Select the current component.
- **Esc**: Close any open menus.

## Tools & Upgrades
- Webpack for bundling assets.
- Babel for JavaScript transpiling.
- Service Workers for offline capabilities.

## Development Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/donkaratu/cloakee.github.io
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```

## Project Structure
- `/src`: Source code files.
- `/public`: Public assets like images and icons.
- `/dist`: Build files for production.
